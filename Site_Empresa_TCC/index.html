Sobre Nossa Empresa


A empresa Alian's Tech surgiu da ideia de quatro amigos que desejavam usar a tecnologia para melhorar o mundo e facilitar a vida de todas as pessoas, o principal objetivo dessa empresa é também tornar o mundo um lugar cada vez mais sustentável. O nosso desejo é que a nossa empresa cresça cada vez mais no ramo da tecnologia e se torne um referência neste mercado.

Nossos Valores


MISSÃO: Prover softwares de acordo com a necessidade do cliente visando sempre à otimização do tempo, a transparência e a qualidade de nosso produto.​

VISÃO: Ser referência em qualidade, velocidade e confiabilidade no ramo de soluções em tecnologia.​

VALORES: Comprometimento, transparência, produtividade, realizar as necessidades do cliente e sustentabilidade.​

Linguagens


Estas são as linguegens que usamos para a criação de nossos projetos:

ADOBE PHOTOSHOP

Adobe Photoshop é um programa profissional de edição de imagem muito famoso, que funciona em Windows, Mac, e tem versão para celular (Androd, iPhone e Windows Phone). Entretando, não existe uma versão para web, ou seja, não é possível usá-lo online. O editor é pago, porém pode ser grátis durante 30 dias, para testes de novos usuários. Ele está diponível em português na maior parte das suas versões.

MYSQL

O MySQL é um sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface. Nele é possível fazer o gerenciamento de todos os dados da sua empresa ou comércio.



PHP

O PHP (um acrônimo recursivo para PHP: Hypertext Preprocessor) é uma linguagem de script open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML.
